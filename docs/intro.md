# 文档总览

丘比特API 是一个面向 AI CLI 工具的统一接入网关。这一页不是安装教程本身，而是帮助你先确认：**你在用什么工具、应该去哪份文档、完成安装后再去哪里管理 Key 和账单**。

## 支持工具

当前文档优先覆盖以下五种工具：

- **Claude Code**：适合长上下文、多文件改动和复杂 Agent 工作流
- **Codex**：适合高频编码、补全和偏 OpenAI 的快速试错场景
- **Gemini CLI**：适合大上下文检索、分析和长文档理解
- **OpenClaw**：偏 Agent / Gateway 形态，适合更完整的工具调用流程
- **OpenCode**：多模型客户端，按你最终所选模型归到对应分组

## 接入步骤

### 1. 先判断你在用哪个工具

如果你已经明确自己在用 Claude Code、Codex、Gemini CLI、OpenClaw 或 OpenCode，就直接进入对应文档；如果还不确定，可以先回首页看工具路径说明。

### 2. 进入对应工具文档完成安装

每份工具文档都已经按平台拆分了安装和配置步骤：

- [Claude Code](/docs/claude-code)
- [Codex](/docs/codex)
- [Gemini CLI](/docs/gemini-cli)
- [OpenClaw](/docs/openclaw)
- [OpenCode](/docs/opencode)

### 3. 安装完成后再进入控制台

控制台负责：

- 注册账户
- 创建和管理 API Key
- 查看调用记录、余额和账单

控制台入口： [QiubiHub 控制台](https://www.qiubithub.com/console)

## 支持模型

QiubiHub 当前支持多种主流模型供应商和兼容路线，包括：

- **OpenAI 系**：GPT 系列、Codex 等兼容模型
- **Anthropic 系**：Claude 系列模型
- **Google 系**：Gemini 系列模型
- **其他兼容模型**：会持续扩展

具体使用哪一条模型路线，最终以你所选的工具文档和配置方式为准。

## 获取帮助

如果你还没开始安装，但已经不确定该走哪条路线：

1. 先回首页看“文档入口 / 工具路径”部分，确认自己应该进入哪份文档
2. 如果仍然拿不准，告诉我们你正在使用的工具和目标模型，我们可以直接帮你判断
