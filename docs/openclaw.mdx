# OpenClaw (Moltbot / Clawdbot)

通过 QiubiHub 插件,在 OpenClaw(Moltbot / Clawdbot)中使用 GPT、Claude 和 Gemini 模型。

:::info 名称变更说明
Clawdbot 改名 Moltbot 然后又改名 Openclaw
:::

---

<Tabs groupId="operating-systems">
  <TabItem value="macos" label="macOS" default>

### 1. 安装 Openclaw

在**终端**中执行:

```bash
npm install -g openclaw@latest
```

安装完成后,验证安装:

```bash
openclaw --version
```

![验证安装](/img/1769757916311-1653af9a-eca5-4d53-a757-1e8826ad2b13.png)

### 2. 安装 QiubiHub 插件

在**终端**中执行:

```bash
openclaw plugins install openclaw-aicodewith-auth
```

![安装插件](/img/1769759405687-80dcb108-e978-4dbe-8d42-e5d88b25fd5d.png)

然后启用插件:

```bash
openclaw plugins enable openclaw-aicodewith-auth
```

![启用插件](/img/1769759499104-bd4ae5cd-2f74-4600-b07b-d001f40e5e5d.png)

:::tip 重启 Gateway
如果 gateway 正在运行,需要重启一下:

```bash
openclaw gateway stop
openclaw gateway run
```
:::

### 3. 配置 API

运行认证命令:

```bash
openclaw models auth login --provider aicodewith-claude --set-default
```

按提示输入你的 **QiubiHub API Key**

![配置API](/img/1769760309877-1d02351d-0089-4399-aabc-5d5afe6ab166.png)

查看可用模型:

```bash
openclaw models list | grep aicodewith
```

![查看模型](/img/1769760489908-b297df82-d6de-458e-86b5-5ad155771d63.png)

## 配置 Telegram

这个东西必须得配置一个跟他对话的地方来使用,目前暂时不支持微信。

:::tip 为什么选择 Telegram?
因为 TG 里面使用体验是最好的,人家创始人就是拿 tg 控制的。建议注册一下,然后手机上挂个梯子就好了。

那些飞书bot、企业微信之类的东西连 "斜杠" 的命令都没有,各种命令都得手敲,体验极其拉垮,非常不建议把时间浪费在那些地方。
:::

### 1. 获取 Telegram Bot Token

1. 在 Telegram 中搜索 `@BotFather`
2. 发送 `/newbot`
3. 按提示设置 bot 名称
4. 获得 Bot Token(格式:`123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

![获取Bot Token](/img/1769524863859-91f7f2d8-acbc-4412-8a4e-aa79b5864ec9.png)

### 2. 启用 Telegram 插件

这个步骤需要在电脑的终端里面执行,这个插件默认是关闭的:

```bash
openclaw plugins enable telegram
```

![启用Telegram插件](/img/1769524968139-d01a1a4f-420f-47a6-8170-f839f05f917b.png)

### 3. 配置 Bot Token

这里输入你第一步获取的那个 token:

```bash
openclaw config set channels.telegram.botToken "你的Bot_Token"
```

![配置Bot Token](/img/1769524959959-39914d50-a9f8-4bb2-aea9-6e77266facde.png)

### 4. 启动 Gateway

```bash
openclaw gateway run
```

如果报错了,可以先尝试停止一下:

```bash
openclaw gateway stop
```

![启动Gateway](/img/1769525461250-dec9b757-ff98-4711-ae1b-657fd3548af8.png)

### 5. 测试

在 Telegram 中找到你的 bot,发送任意消息,bot 会给你一个验证码:

![Telegram验证码](/img/1769525324221-d3cd4443-5abd-4bc0-9fcc-a8698e028385.jpg)

然后你需要在电脑上执行下面的命令:

```bash
openclaw pairing approve telegram 你的验证码
```

![配对成功](/img/1769525412735-43273a57-a1ee-4da1-8f01-0c01622cf011.png)
