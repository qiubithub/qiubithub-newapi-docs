# 常见问题

## 账户相关

### 如何注册账户？

访问 [qiubithub.com](https://qiubithub.com)，点击右上角「注册」按钮，填写邮箱和密码即可。

### 忘记密码怎么办？

在登录页面点击「忘记密码」，输入注册邮箱，我们会发送重置链接到你的邮箱。

### 如何修改邮箱？

目前不支持直接修改邮箱。如需更换，请联系客服处理。

### 可以注销账户吗？

可以。请确保账户余额为 0，然后发送邮件至 support@qiubithub.com 申请注销。

## API Key 相关

### API Key 泄露了怎么办？

立即在控制台删除该 Key，然后创建新的 Key 并更新你的应用配置。

### 可以创建多个 API Key 吗？

可以。建议为不同项目或环境创建独立的 Key，便于管理和监控。

### API Key 有有效期吗？

没有。除非手动删除，否则永久有效。

### 如何限制 API Key 的权限？

创建 Key 时可设置：
- 可用模型范围
- 请求频率限制
- IP 白名单

## 计费相关

### 免费额度用完了怎么办？

充值后即可继续使用。支持支付宝和微信支付。

### 充值后多久到账？

即时到账。支付成功后立即可用。

### 余额可以退款吗？

充值 7 天内且未使用可申请退款。联系客服处理。

### 如何查看详细账单？

登录控制台，进入「账单中心」-「消费记录」查看详细账单。

## 技术相关

### 支持哪些编程语言？

所有语言都支持。通过 HTTP API 调用，任何能发起 HTTP 请求的语言都可以使用。

### 有 SDK 吗？

可以使用官方 SDK，只需修改 base_url：
- Python: `openai` 库
- Node.js: `openai` 包
- Go: `github.com/sashabaranov/go-openai`

### 遇到 429 错误怎么办？

表示请求过于频繁。解决方案：
1. 降低请求频率
2. 升级套餐提高限制
3. 使用指数退避重试

### 遇到 402 错误怎么办？

表示余额不足。请充值后再试。

### 最大请求体大小是多少？

默认 10MB。如需更大限制，请联系客服。

## 模型相关

### 如何选择合适的模型？

| 任务类型 | 推荐模型 |
|---------|---------|
| 日常对话 | GPT-4o-mini |
| 代码生成 | Claude 3.5 Sonnet |
| 复杂推理 | GPT-4o / Claude 3 Opus |
| 长文档 | Gemini 1.5 Pro |
| 快速响应 | Claude 3 Haiku |

### 模型会更新吗？

会的。我们会及时跟进官方更新，并在控制台公告。

### 可以微调模型吗？

目前不支持。如需此功能，请联系我们讨论企业方案。
